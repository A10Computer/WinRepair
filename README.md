A modern **Windows repair and system maintenance toolkit** built with PowerShell and Windows Forms.

It provides a simple graphical interface to execute common administrative, repair, and diagnostic tasks with **one click and automatic admin elevation**.

---

## 🧩 Features

### 🛡️ System Repair
- 🔍 SFC System File Check (`sfc /scannow`)
- 🧪 SFC Verification Only
- 🧰 DISM CheckHealth (quick system image check)
- 🔧 DISM ScanHealth (deep analysis)
- 🛠️ DISM RestoreHealth (system repair)
- 🧹 Component Store Cleanup
- ⚡ Deep Cleanup with ResetBase option

---

### 🌐 Network Tools
- 🧭 ARP Cache Reset
- 🌍 DNS Flush
- 🔥 Windows Firewall Reset
- 🔄 Winsock & TCP/IP Reset
- 📡 IP Renew (Release / Renew)

---

### 🧹 Cleanup Tools
- 🗑️ Disk Cleanup (cleanmgr)
- 📧 Outlook Cache Cleanup
- 💬 Microsoft Teams Cache Cleanup

---

### 🧰 Windows Administration
- 🖥️ Device Manager
- 🧾 Registry Editor
- ⚙️ Services Manager
- 🧩 System Configuration (msconfig)
- 🔄 Windows Update Settings

---

### 🚀 Advanced Tools
- 💽 CHKDSK Disk Check
- 🔧 Windows Update Repair (reset services + components)

---

## ⚙️ Requirements

- 💻 PowerShell 5.1 or higher
- 🔑 Administrator privileges (required for most functions)
