Windows Repair Tools is a compact PowerShell-based GUI utility for Windows 10 and Windows 11. It brings common repair, diagnostic, cleanup, network, and system administration actions together in one simple interface.

The tool is designed for technicians, power users, and anyone who wants quick access to frequently used Windows maintenance functions without typing commands manually.

Features: SFC and DISM repair/check actions, Windows Update repair/reset, Network repair tools such as DNS flush, IP renew, proxy reset, firewall reset, and TCP/IP reset

Event Viewer and Reliability Monitor shortcuts, CBS log viewer for SFC/DISM troubleshooting, Disk cleanup and cache cleanup for Outlook and Teams

Print Spooler reset, Restore point creation and System Protection access

Quick access to Device Manager, Disk Management, Services, Task Scheduler, Group Policy Editor, Registry Editor, and more


Requirements
Windows 10 or Windows 11

Administrator rights are required for many repair and reset functions

Some tools may only be available on specific Windows editions, for example Group Policy Editor on Pro/Enterprise editions



Important Notice

Use this tool at your own risk. Some actions can modify system settings, clear logs, reset network configuration, or require a restart. Always create a restore point before running repair actions if possible.
